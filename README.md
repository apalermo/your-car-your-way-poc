# Your Car Your Way - PoC Tchat Temps Réel

Bienvenue sur le dépôt du Proof of Concept de la fonctionnalité de messagerie temps réel.

## 🏗️ Architecture du Monorepo

Ce projet est divisé en deux applications distinctes :

- 🔙 **[Backend (Spring Boot & WebSockets)](./backend/README.md)** : Gère la logique métier, la persistance PostgreSQL et le broker de messages STOMP.
- 🔜 **[Frontend (Angular)](./frontend/README.md)** : Interface utilisateur pour tester la communication bidirectionnelle en temps réel. _(À venir)_

👉 **Pour lancer le projet, veuillez consulter les instructions spécifiques dans le README de chaque dossier.**
